# How-to-README
วิธีการใช้งาน README
## หัวข้อ(Headings)
```
# หัวข้อที่ 1
## หัวข้อที่ 2
### หัวข้อที่ 3
```
# หัวข้อที่ 1
## หัวข้อที่ 2
### หัวข้อที่ 3

## อักษรตัวหนา(Bold)
```
**ตัวหนา**
```
อักษร**ตัวหนา**

## อักษรตัวเอียง(Italic)
```
_ตัวเอียง_
```
อักษร _ตัวเอียง_

## ขีดเส้นกลาง(Strikethrough)
```
~~ขึดเส้นกลาง~~
```
ตัวอักษร~~ขีดเส้นกลาง~~
